para iniciantes

Um curso incrível, presencial e totalmente gratuito.
Com seis aulas que misturam técnicas de aprendizado, competições e prêmios, este é com certeza o melhor curso para você aprender a arte de criação de sites, e se tornar o próximo Web Developer Full-Stack.

O curso começa em 26/04/2016.

Se você quer muito participar mas não se inscreveu, mande-me um email.

Primeira aula

Essas imagens apenas representam o perfil da primeira aula.

O curso traçará desde o começo todos os aspectos mais importantes para se tornar um bom desenvolvedor web.

Você vai entender a fundo, de uma forma interativa e de maneira ampla as melhores práticas que as melhores empresas e os melhores desenvolvedores usam na hora de criar conteúdo para a internet.

E muito mais!

Serão seis aulas ao todo, com duração entre 1h e 1h30m, espalhadas ao longo de 3 a 4 semanas.

O curso será ministrado na UFMG, campus Pampulha, mas é aberto a todas as pessoas interessadas.

A data ainda não foi definida. Para isso (e para outras questões, como escolha de sala), pedimos sua ajuda! Se inscreva no formulário abaixo caso esteja interessado, informando algumas coisas, como a sua disponibilidade de horário e dias preferidos.

Programação

  1. Como a Internet funciona e como você pode usá-la
  2. HTML nível 2, CSS nível 1 e a sua primeira página Web
  3. Javascript nível 2 + competição em grupo
  4. Seu primeiro site quase completo + prêmio de criatividade
  5. HTML nível 4 e Javascript nível 4 - Conceitos para alta performance
  6. (bônus) CSS nível 3 e ferramentas para Desenvolvimento Ágil
Os níveis mencionados têm como referência o nível 5 como o máximo.

O curso não tem como foco o CSS, isso é, o design/layout da página, por isso será um tema menos abordado.

O curso não chega ao nível 5 pois acredito que a excelência no trabalho de qualquer pessoa vem com a prática. Portanto, depende de cada um se esforçar para se manter atualizado com as novas tecnologias e sempre correr atrás de evolução. Algumas técnicas e boas práticas, eu tentarei passar durante esse período de convivência, para te guiar na sua trajetória web no futuro.
